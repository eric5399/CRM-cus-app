## 二、文字系统

### 2.1 字体族

| 类型 | 字体 | 说明 |
|------|------|------|
| 中文 | PingFang SC | 系统默认中文字体 |
| 英文 | Helvetica Neue | 英文及部分符号 |
| 数字 | D-DIN-PRO | 数字和金额显示 |

### 2.2 字重

| 字重 | 等级 |
|------|------|
| Bold | 700 |
| Medium/Semibold | 500-600 |
| Regular | 400 |

### 2.3 字体系统详细定义

| 编码 | 字号/行高 | 字体 | 字重 | 使用场景 |
|------|----------|------|------|---------|
| T0 | 40px/56px | PingFang SC | Medium | 瀑布流分类标签选中 |
| T1 | 36px/50px | PingFang SC | Medium | 弹窗标题 |
| T2 | 34px/48px | PingFang SC | Medium | 资讯标题、大按钮文字、顶部导航栏标题 |
| T3 | 32px/44px | PingFang SC | Regular | 信息列表标题、主要文本信息、小标题 |
| T4 | 28px/40px | PingFang SC | Regular | 小按钮文本、次要文本信息、弹窗正文、列表内容 |
| T5 | 26px/36px | PingFang SC | Regular | 个人主页功能入口 |
| T6 | 24px/34px | PingFang SC | Regular | 说明、提示等辅助性文字、跳转链接 |
| T7 | 22px/30px | PingFang SC | Regular | 运营提醒气泡、标签文字 |
| T8 | 20px/28px | PingFang SC | Regular | 标签文字、底部tab、最小字号 |

**备注：** 
- 默认行高为字号的 1.5 倍（可在此基础上略微调整）
- 字间距统一设置为 0
- iOS 和 Android 的栅格系统间隙都是 2 的倍数，字号都是 4 的倍数

## 三、全局基础规范

### 3.1 界面尺寸与布局

| 项目 | 数值 | 说明 |
|------|------|------|
| UI稿尺寸 | 750px × 1624px | 标准移动端设计稿尺寸 |
| 页边距 | 24px | 左右固定边距 |
| 模块间距 | 20px | 模块与模块之间的间距（优先） |
| 模块间距备选 | 16px 或 18px | 特殊场景中可根据页宽动态调整，但必须为复数 |
| 栅格基数 | 4px 的倍数 | 界面间隙须为 4 的倍数 |

### 3.2 圆角规范

| 应用场景 | 圆角 |
|---------|------|
| 大卡片 | 24px |
| 内部卡片、按钮、输入框等 | 12px |

### 3.3 描边规范

| 场景 | 颜色 | 宽度 |
|------|------|------|
| 分隔线（灰色背景） | #D8DAE3 | 1px |
| 分隔线（白色背景） | #E8ECF2 | 1px |

### 3.4 布局占比（多列布局）

| 列数 | 宽度占比 | 说明 |
|------|---------|------|
| 两列 | 45.6% | 内容为主的两列布局 |
| 三列 | 29.6% | 均衡三列布局 |
| 四列 | 21.6% | 紧凑四列布局 |

**备注：** 页面布局最多四列，可滑动组件无边距

## 五、控件组件规范

### 5.1 按钮规范

**圆角：** 12px  
**最小宽度：** 根据内容，最少容纳四字文案

**状态定义：**

| 状态 | 背景色 | 文字色 | 说明 |
|------|--------|--------|------|
| Default（默认） | #FF6611 | #FFFFFF | 正常可交互状态 |
| Hover（悬浮） | #CC4E0A | #FFFFFF | 用户悬浮时状态 |
| Active（点击） | #993A06 | #FFFFFF | 用户按下时状态 |
| Disabled（禁用） | #D8DAE3 | #AFB3C2 | 不可交互状态 |

### 5.2 输入框规范

**圆角：** 12px  
**高度：** 56px（单行输入）  
**边框宽度：** 1px

**状态定义：**

| 状态 | 边框色 | 背景色 | 文字色 |
|------|--------|--------|--------|
| Default（默认） | #D8DAE3 | #FFFFFF | #222222 |
| Focus（聚焦） | #FF6611 | #FFFFFF | #222222 |
| Filled（已填写） | #E8ECF2 | #FFFFFF | #222222 |
| Error（错误） | #E85757 | #FFEAE9 | #E85757 |
| Disabled（禁用） | #E8ECF2 | #F4F6F9 | #AFB3C2 |
| ReadOnly（只读） | #E8ECF2 | #F4F6F9 | #737587 |

**占位符文字色：** #AFB3C2

### 5.3 Radio（单选框）规范

**大小：** 44px × 44px  
**内圆点大小：** 根据选中状态

**状态定义：**

| 状态 | 外圆色 | 内圆点 | 文字色 |
|------|--------|--------|--------|
| Default（未选） | #D8DAE3 | 无 | #222222 |
| Hover（悬浮） | #AFB3C2 | 无 | #222222 |
| Selected（已选） | #FF6611 | #FFFFFF | #222222 |
| Disabled（禁用） | #E8ECF2 | 无 | #AFB3C2 |

### 5.4 Checkbox（多选框）规范

**大小：** 44px × 44px  
**圆角：** 4px

**状态定义：**

| 状态 | 背景色 | 勾选色 | 文字色 |
|------|--------|--------|--------|
| Default（未选） | #FFFFFF | - | #222222 |
| Hover（悬浮） | #F4F6F9 | - | #222222 |
| Selected（已选） | #FF6611 | #FFFFFF | #222222 |
| Indeterminate（半选） | #FF6611 | #FFFFFF | #222222 |
| Disabled（禁用） | #E8ECF2 | - | #AFB3C2 |

### 5.5 开关/Toggle规范

**大小：** 56px × 32px  
**圆角：** 16px（胶囊形）

**状态定义：**

| 状态 | 背景色 | 滑块色 | 说明 |
|------|--------|--------|------|
| Off（关闭） | #D8DAE3 | #FFFFFF | 未启用状态 |
| On（开启） | #56C468 | #FFFFFF | 启用状态 |
| Disabled（禁用） | #E8ECF2 | #AFB3C2 | 不可交互 |

### 5.6 标签/Tag规范

**圆角：** 12px  
**高度：** 32px  
**内边距：** 8px（左右）

**类型：**

| 类型 | 背景色 | 文字色 | 边框 | 用途 |
|------|--------|--------|------|------|
| 普通标签 | #F4F6F9 | #222222 | 1px #D8DAE3 | 分类、属性标记 |
| 功能标签 | #FF6611 | #FFFFFF | 无 | 重要标记、活动标签 |
| 成功标签 | #56C468 | #FFFFFF | 无 | 成功状态标记 |
| 警告标签 | #F7B44C | #FFFFFF | 无 | 警告、待处理 |
| 错误标签 | #E85757 | #FFFFFF | 无 | 错误、失败状态 |

## 六、组件命名规则

### 6.1 命名约定

**格式：** `类别/子类别/状态/大小`

**示例：**
- `Button/Primary/Default/Large`
- `Input/Text/Focus/Standard`
- `Card/Product/Default`

### 6.2 常用类别

| 类别 | 子类别 | 说明 |
|------|--------|------|
| Button | Primary、Secondary、Tertiary | 按钮组件 |
| Input | Text、Number、Password、TextArea | 输入框组件 |
| Checkbox | Default、Group | 多选框 |
| Radio | Default、Group | 单选框 |
| Toggle | Default | 开关 |
| Tag | Default、Outline、Filled | 标签 |
| Card | Product、News、Profile、Custom | 卡片 |

## 七、交互状态矩阵

### 7.1 完整状态覆盖

所有交互组件应包含以下状态：

| 状态 | 说明 | 应用场景 |
|------|------|---------|
| Default | 默认/初始状态 | 页面初始加载 |
| Hover | 鼠标悬浮状态 | 桌面端交互 |
| Active/Pressed | 按下/点击状态 | 用户触发操作时 |
| Focus | 焦点/聚焦状态 | 键盘导航、输入框获焦 |
| Disabled | 禁用状态 | 不可操作场景 |
| Loading | 加载中状态 | 异步操作进行中 |
| Error | 错误状态 | 校验失败、错误提示 |
| Success | 成功状态 | 操作成功反馈 |

## 八、栅格系统规范

### 8.1 基础栅格

- **基础单位：** 4px
- **所有间距必须：** 4px 的倍数
- **所有字号必须：** 4px 的倍数

### 8.2 常用间距值

| 间距 | 用途 |
|------|------|
| 4px | 极小间距 |
| 8px | 小间距 |
| 12px | 中小间距 |
| 16px | 标准间距 |
| 20px | 大间距 |
| 24px | 页边距 |
| 32px | 特大间距 |
